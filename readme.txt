to initial it as git repo
git init
to add repo to local
git remote add origin "paste here"
git branch -m master

to push code to git 
git add . 
git commit -m "message"
git push -u origin master

to pull
git pull origin master
to get all changes 
git pull

to see logs 
git log

to see the user name  of git 
git config --global user.name

to see the user email  of git 
git config --global user.email
